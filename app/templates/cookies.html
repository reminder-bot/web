Cookies and Session Data

This website uses some cookies and some session data. None of these can be disabled since they are necessary to operate the site. The data this site stores locally is not shared to any other sites. Session data is signed to prevent modification.

The dashboard on this website uses cookies and session data to operate

Cookies

We store the following cookies:

  • meridian: Whether times are to be formatted in 12 hour or 24 hour
  • guild-order: The order of the guilds in the sidebar (drag-and-drop to rearrange them)
  • asked-cookies: Set if you've already closed the cookie warning

Session Data

We store the following session data:

  • Discord OAuth tokens necessary for querying the user ID and the guilds from your Discord.
  • Internal ID used to modify or reference your user when you make requests to our API
